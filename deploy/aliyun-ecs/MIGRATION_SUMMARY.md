## 详细改动说明

### 改动1: Terraform Provider

**难度**: ⭐ 简单

**AWS版本**:
```hcl
provider "aws" {
  region = var.aws_region
}
```

**阿里云版本**:
```hcl
provider "alicloud" {
  region     = var.region
  access_key = var.access_key
  secret_key = var.secret_key
}
```

### 改动2: VPC网络配置

**难度**: ⭐⭐ 中等

需要理解阿里云的VPC模型：
- AWS: VPC → Subnet
- 阿里云: VPC → VSwitch

### 改动3: 存储服务调用

**难度**: ⭐⭐ 中等

**AWS S3**:
```python
import boto3
s3 = boto3.client('s3')
s3.upload_fileobj(file, bucket, key)
```

**阿里云OSS**:
```python
import oss2
auth = oss2.Auth(access_key, secret_key)
bucket = oss2.Bucket(auth, endpoint, bucket_name)
bucket.put_object(key, file)
```
