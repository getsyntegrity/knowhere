# Code Of Conduct

This project is committed to a respectful, harassment-free community for
everyone.

## Expected Behavior

- Be respectful in discussions, reviews, and issue triage.
- Focus feedback on the code, design, and documented behavior.
- Assume good intent while still being direct about technical concerns.
- Help keep the repository welcoming to contributors with different
  backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, discrimination, threats, or abusive language.
- Publishing private information without permission.
- Personal attacks, intimidation, or disruptive trolling.
- Any conduct that would make participation unsafe or hostile.

## Reporting

If you experience or witness unacceptable behavior, report it privately to
the repository maintainers instead of opening a public issue about the
incident.

## Enforcement

Project maintainers may remove, edit, or reject contributions, comments, or
other interactions that violate this Code of Conduct.
