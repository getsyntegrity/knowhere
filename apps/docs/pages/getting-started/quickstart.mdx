# 快速开始

本指南将帮助您在几分钟内开始使用 Knowhere AI 知识库管理系统。

## 前提条件

- 一个 Knowhere AI 账户
- Python 3.7+ 或 Node.js 16+
- 基本的编程知识

## 步骤 1: 创建账户和获取 API Key

1. 访问 [Knowhere AI 控制台](https://knowhere.ai/dashboard)
2. 使用 Google、GitHub 或 Apple 登录
3. 在控制台中创建 API Key
4. 保存您的 API Key（稍后需要用到）

## 步骤 2: 安装 SDK

### Python

```bash
pip install knowhere-sdk
```

### TypeScript/JavaScript

```bash
npm install @knowhere/sdk-typescript
```

## 步骤 3: 创建您的第一个知识库

### Python 示例

```python
from knowhere_sdk import KnowhereClient

# 初始化客户端
client = KnowhereClient(api_key="your-api-key-here")

# 创建知识库任务
job = client.kb.create_job(
    file_url="https://example.com/sample-document.pdf",
    webhook_url="https://your-app.com/webhook"  # 可选
)

print(f"任务已创建: {job.job_id}")
print(f"任务状态: {job.status}")

# 检查任务状态
status = client.kb.get_job_status(job.job_id)
print(f"当前状态: {status.current_state}")
```

### TypeScript 示例

```typescript
import { KnowhereClient } from '@knowhere/sdk-typescript';

// 初始化客户端
const client = new KnowhereClient({
  apiKey: 'your-api-key-here'
});

// 创建知识库任务
const job = await client.kb.createJob({
  fileUrl: 'https://example.com/sample-document.pdf',
  webhookUrl: 'https://your-app.com/webhook'  // 可选
});

console.log(`任务已创建: ${job.jobId}`);
console.log(`任务状态: ${job.status}`);

// 检查任务状态
const status = await client.kb.getJobStatus(job.jobId);
console.log(`当前状态: ${status.currentState}`);
```

## 步骤 4: 设置 Webhook（可选）

为了实时接收任务状态更新，您可以设置 Webhook：

```python
# 配置 Webhook
webhook_config = client.webhook.create_config(
    webhook_url="https://your-app.com/webhook",
    events=["job.completed", "job.failed"]
)

print(f"Webhook 已配置: {webhook_config.webhook_id}")
```

## 常见问题

### Q: 支持哪些文件格式？
A: 我们支持 PDF、DOCX、XLSX、PPTX、MD、TXT 等多种格式。

### Q: 任务处理需要多长时间？
A: 处理时间取决于文件大小和复杂度，通常几分钟到几小时不等。

### Q: 如何获取任务状态？
A: 您可以通过 API 轮询状态，或设置 Webhook 接收实时通知。

### Q: 有使用限制吗？
A: 免费用户每月有 100 个 Credits，付费用户有更高的限制。
