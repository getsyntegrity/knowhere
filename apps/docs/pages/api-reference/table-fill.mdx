# 表格填充 API

表格填充 API 允许您上传表格文件，基于知识库内容智能填充表格数据。

## 基础信息

- **基础 URL**: `https://api.knowhere.ai`
- **API 版本**: v1
- **认证**: Bearer Token (API Key)

## 创建表格填充任务

创建一个新的表格填充任务。

### 请求

```http
POST /v1/table-fill/jobs
Content-Type: application/json
Authorization: Bearer <YOUR_API_KEY>

{
  "file_url": "https://example.com/data.xlsx",
  "webhook_url": "https://your-app.com/webhook",
  "metadata": {
    "description": "Q4销售数据填充"
  }
}
```

### 参数

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `file_url` | string | 是 | 表格文件的 URL |
| `webhook_url` | string | 否 | Webhook 通知 URL |
| `metadata` | object | 否 | 额外的元数据 |

### 响应

```json
{
  "success": true,
  "data": {
    "job_id": "job_1234567890",
    "status": "pending",
    "current_state": "uploading_file",
    "created_at": "2024-01-15T10:30:00Z",
    "file_url": "https://example.com/data.xlsx",
    "webhook_url": "https://your-app.com/webhook"
  }
}
```

## 查询任务状态

获取表格填充任务的当前状态。

### 请求

```http
GET /v1/table-fill/jobs/{job_id}
Authorization: Bearer <YOUR_API_KEY>
```

### 路径参数

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `job_id` | string | 是 | 任务 ID |

### 响应

```json
{
  "success": true,
  "data": {
    "job_id": "job_1234567890",
    "status": "processing",
    "current_state": "extracting_table",
    "progress": 45,
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:32:00Z",
    "file_url": "https://example.com/data.xlsx",
    "webhook_url": "https://your-app.com/webhook",
    "metadata": {
      "description": "Q4销售数据填充"
    }
  }
}
```

### 状态说明

| 状态 | 说明 |
|------|------|
| `pending` | 任务已创建，等待处理 |
| `processing` | 任务正在处理中 |
| `completed` | 任务已完成 |
| `failed` | 任务处理失败 |

### 详细状态

| 状态 | 说明 |
|------|------|
| `uploading_file` | 正在上传文件 |
| `extracting_table` | 正在提取表格 |
| `retrieving_kb` | 正在检索知识库 |
| `llm_organizing` | LLM 正在整理数据 |
| `filling_table` | 正在填充表格 |
| `generating_result` | 正在生成结果文件 |

## 下载结果

下载表格填充的结果文件。

### 请求

```http
GET /v1/table-fill/jobs/{job_id}/download
Authorization: Bearer <YOUR_API_KEY>
```

### 路径参数

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `job_id` | string | 是 | 任务 ID |

### 响应

成功时返回文件流，失败时返回错误信息：

```json
{
  "success": false,
  "error": {
    "code": "job_not_completed",
    "message": "任务尚未完成"
  }
}
```

## 错误处理

### 常见错误

| 错误码 | HTTP 状态码 | 说明 |
|--------|-------------|------|
| `invalid_file_url` | 400 | 文件 URL 无效 |
| `unsupported_file_type` | 400 | 不支持的文件类型 |
| `job_not_found` | 404 | 任务不存在 |
| `job_not_completed` | 400 | 任务尚未完成 |
| `insufficient_credits` | 402 | Credits 不足 |

### 错误响应格式

```json
{
  "success": false,
  "error": {
    "code": "error_code",
    "message": "错误描述",
    "details": "详细错误信息"
  }
}
```

## 使用示例

### Python

```python
import requests
import time

# 创建任务
response = requests.post(
    "https://api.knowhere.ai/v1/table-fill/jobs",
    headers={"Authorization": "Bearer <YOUR_API_KEY>"},
    json={
        "file_url": "https://example.com/data.xlsx",
        "webhook_url": "https://your-app.com/webhook"
    }
)

job = response.json()["data"]
print(f"任务已创建: {job['job_id']}")

# 轮询状态
while True:
    status_response = requests.get(
        f"https://api.knowhere.ai/v1/table-fill/jobs/{job['job_id']}",
        headers={"Authorization": "Bearer <YOUR_API_KEY>"}
    )
    
    status = status_response.json()["data"]
    print(f"当前状态: {status['current_state']}")
    
    if status["status"] in ["completed", "failed"]:
        break
    
    time.sleep(5)

# 下载结果
if status["status"] == "completed":
    download_response = requests.get(
        f"https://api.knowhere.ai/v1/table-fill/jobs/{job['job_id']}/download",
        headers={"Authorization": "Bearer <YOUR_API_KEY>"}
    )
    
    with open("result.xlsx", "wb") as f:
        f.write(download_response.content)
    
    print("结果文件已下载")
```

### JavaScript

```javascript
// 创建任务
const createJob = async () => {
  const response = await fetch('https://api.knowhere.ai/v1/table-fill/jobs', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer <YOUR_API_KEY>',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      file_url: 'https://example.com/data.xlsx',
      webhook_url: 'https://your-app.com/webhook'
    })
  });
  
  const job = await response.json();
  console.log(`任务已创建: ${job.data.job_id}`);
  return job.data;
};

// 轮询状态
const pollStatus = async (jobId) => {
  while (true) {
    const response = await fetch(`https://api.knowhere.ai/v1/table-fill/jobs/${jobId}`, {
      headers: {
        'Authorization': 'Bearer <YOUR_API_KEY>'
      }
    });
    
    const status = await response.json();
    console.log(`当前状态: ${status.data.current_state}`);
    
    if (['completed', 'failed'].includes(status.data.status)) {
      return status.data;
    }
    
    await new Promise(resolve => setTimeout(resolve, 5000));
  }
};

// 下载结果
const downloadResult = async (jobId) => {
  const response = await fetch(`https://api.knowhere.ai/v1/table-fill/jobs/${jobId}/download`, {
    headers: {
      'Authorization': 'Bearer <YOUR_API_KEY>'
    }
  });
  
  if (response.ok) {
    const blob = await response.blob();
    const url = window.URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'result.xlsx';
    a.click();
    console.log('结果文件已下载');
  }
};

// 完整流程
const main = async () => {
  const job = await createJob();
  const status = await pollStatus(job.job_id);
  
  if (status.status === 'completed') {
    await downloadResult(job.job_id);
  }
};
```

## 限制和配额

- 文件大小限制：100MB
- 支持格式：XLSX, XLS, CSV
- 处理超时：2小时
- 并发任务限制：根据订阅计划而定
