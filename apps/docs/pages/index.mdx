# Knowhere AI 文档

欢迎使用 Knowhere AI 知识库管理系统！这是一个基于 AI 的智能知识库管理和问答系统，提供强大的文档解析、向量化存储和智能检索功能。

## 🚀 快速开始

### 1. 获取 API Key

首先，您需要在 [控制台](https://knowhere.ai/dashboard) 创建一个账户并获取 API Key。

### 2. 安装 SDK

选择您喜欢的语言：

```bash
# Python
pip install knowhere-sdk

# TypeScript/JavaScript
npm install @knowhere/sdk-typescript
```

### 3. 创建您的第一个知识库

```python
from knowhere_sdk import KnowhereClient

client = KnowhereClient(api_key="your-api-key")

# 创建知识库任务
job = client.kb.create_job(
    file_url="https://example.com/document.pdf",
    webhook_url="https://your-app.com/webhook"
)

print(f"任务已创建: {job.job_id}")
```

## ✨ 核心功能

### 📊 表格填充
智能表格数据填充，基于知识库内容自动填充表格字段。

- 支持多种文件格式（PDF、DOCX、XLSX等）
- 智能表格识别和提取
- 基于知识库的智能填充
- 异步处理，支持Webhook通知

### 🧠 知识库管理
强大的文档解析和向量化存储系统。

- 多格式文档解析（PDF、DOCX、PPTX、MD、TXT等）
- 智能文档分块和向量化
- 向量数据库存储
- 实时状态追踪

### 🔗 Webhook 集成
实时任务状态通知，支持自定义Webhook URL。

- HMAC-SHA256 签名验证
- 指数退避重试机制
- 完整的日志记录
- 幂等性保证

## 💡 最佳实践

1. **合理设置 Webhook URL**：确保您的 Webhook 端点能够快速响应
2. **监控任务状态**：定期检查任务状态，处理失败的任务
3. **优化文档质量**：上传高质量的文档以获得更好的解析效果
4. **使用适当的文件格式**：PDF 和 DOCX 通常有更好的解析效果
